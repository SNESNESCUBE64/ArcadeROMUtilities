# ArcadeROMUtilities
This contains utilities that I have had to write for Free Play or Other Mods. These were mainly used for patching checksums or encrypting or decrypting ROMs. 

## Tools / Function Description
### Pacman Plus Encryption Utility
This tool was used to encrypt and decrypt the ROM data. When you decrypt the data, you can then see the binaries as the CPU sees it. This is great for editing ROM data for mod purposes.
